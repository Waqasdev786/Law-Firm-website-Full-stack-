# Law-Firm-website-Full-stack-
Full-stack lawyers firm website built with Laravel - CRUD operations, Laravel Breeze authentication, MySQL database &amp; responsive Bootstrap frontend.


##  Tech Stack
- Frontend: HTML, CSS, Bootstrap, JavaScript
- Backend: PHP, Laravel 
- Database: MySQL
- Auth: Laravel Breeze

##  Features
- Responsive frontend
- Client registration form
- Admin panel (CRUD)
- Login / Logout system
- Protected routes

##  Admin Login
- URL: /login
- Email: admin@lawyers.com
- Password: admin123

##  Setup
```bash
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
